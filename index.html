東京広域電話網とは？

東京広域電話網（Tokyo Wide Area Telephony Network）とは、IP 電話技術を利用したオレオレ電話網です。

東京広域電話網について発表した動画があります: 令和に黒電話とワープロで遊ぶオタク達 エンジニア作業飲み集会 2025/02/21

Mantela

Mantela は電話交換局の相互接続状況を記述するための JSON Schema です。 Mantela を利用した各種アプリケーションが開発されています。

Mantela Viewer
東京広域電話網の現在の様相を視覚的に確認できます
scrwnl/mikopbx-mantela
MikoPBX の設定データベースから Mantela のテンプレートを生成できます
Mantela Editor
Web アプリケーションによる作成支援を受けながら Mantela を作成できます
Mantela Router
東京広域電話網の電話番号を自動検索できます
Mantela Fetcher
Mantela を一括で効率的に取得できます
Mantela Claimer
各局によって予約されている推奨プレフィックス番号を一覧できます
Mantela Printer for PostScript
各局が持つ局番を PostScript 形式で出力し、実質的な電話帳として利用できるようにします
Mantela Mapper
局あるいは端末の位置情報を地理院地図上に表示します
Mantela Printer for LaTeX
各局が持つ局番を LaTeX のソースコードとして出力し、これをコンパイルすることで実質的な電話帳として利用できるようにします

よくある質問

電話網に参加するにはどうしたらいいですか？

いまさら VoIP 網 などを参考に IP 電話サーバを設置する必要があります。 単に電話したいだけなら Web 電話アプリ を利用することもできます。

電話網に参加したいけど黒電話がありません

必要ありません（えっ）。 Linphone などの IP 電話クライアントソフトウェアを用いて電話網に接続できます。

この VoIP ルータで電話網に接続できますか？

以下の VoIP ルータで電話網に接続できることを確認しています。 ただし、以下の各製品の利用を推奨するものではありません。

  • YAMAHA RT57i
  • YAMAHA RT58i
  • YAMAHA NVR500
  • ICOM VE-TA10（ただし、インチキが必要）
  • YAMAHA RT56v
  • Brother MIP-3020
  • AudioCodes MP-118（ただし、ダイヤル電話では設定が必要、また各種信号は日本のものと異なる）

この VoIP フォンで電話網に接続できますか？

以下の VoIP フォンで電話網に接続できることを確認しています。 ただし、以下の各製品の利用を推奨するものではありません。

  • snom technology snom 220
  • TelePhoSee WVP-2100 Series
  • Grandstream GXP1625
  • 株式会社ナカヨ IP-24N-ST101A

いづれ日本広域電話網や世界広域電話網になりますか？

どうでしょうね。 東京広域電話網はすでに名が体を表しておりません。 電話局と電話機との分布を それぞれ地図のマゼンタとシアンとの塗り潰しで示します。
